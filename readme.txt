                                                Instalace a spuštění aplikace
Spuštění pomocí YARN:
---------------------
1. Přejděte do složky ./todoplus/src
2. Pomocí terminálu zadejte příkaz "yarn install"
3. Následně zadejte příkaz "yarn start"
4. Vyčkejte na spuštění aplikace
