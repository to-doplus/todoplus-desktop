                                                Instalace a spuštění aplikace
Ubuntu:
---------
Standartní instalace:
1. Stáhněte si instalační balíček a přejděte do složky s instalátorem
2. Dvakrát klikněte na soubor "todoplus_1.0.0_amd64.deb", který otevře instalační okno
3. Zvolte tlačítko Install a vyčkejte na stažení

Instalace pomocí terminálu:
1. Stáhněte si instalační balíček a pomocí terminálu přejděte do složky s instalátorem
2. V této složce zadejte příkaz "sudo apt install ./todoplus_1.0.0_amd64.deb" a vyčkejte dokud nebudou staženy všechny potřebné soubory

Spuštění:
Program lze spusti vyhledáním v aplikačním menu nebo zadáním příkazu "todoplus" přímo v terminálu.
